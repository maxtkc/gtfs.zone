    Design Considerations

    • Easy Same Time: Most stops have same arrival/departure, so this should be the default
    • Visual Distinction: Arrival (blue) vs Departure (orange) color coding
    • Propagation: Easy way to apply time changes to multiple following stops
    • Compact: Cells should be reasonably sized for timetable view
    • Progressive Enhancement: Start simple, reveal complexity when needed
    • Keyboard Navigation: Tab, arrow keys, Enter to save
